# Pomodoro for Playdate

番茄钟，由 [Pomodoro for Playdate](https://github.com/risolvipro/playdate-pomodoro) 移植而来。

## 功能介绍

### 时钟首页

默认显示当前时间和日期。

<p align="center">
  <img src="assets/images/clock.png" alt="时钟首页" width="400">
</p>

### 工作计时

提供默认 25 分钟的专注计时。

<p align="center">
  <img src="assets/images/worktime.png" alt="工作计时" width="400">
</p>

### 休息计时

提供默认 5 分钟的休息计时。

<p align="center">
  <img src="assets/images/breaktime.png" alt="休息计时" width="400">
</p>

### 循环番茄钟

可按「工作 → 休息」的节奏自动循环，默认循环 3 次。

### 自定义设置

支持调整循环次数、工作时长和休息时长。

<p align="center">
  <img src="assets/images/settings.png" alt="自定义设置" width="400">
</p>

### 主题切换

支持浅色和深色两种显示主题。

<p align="center">
  <img src="assets/images/theme-light.png" alt="浅色主题" width="400">
</p>

<p align="center">
  <img src="assets/images/theme-dark.png" alt="深色主题" width="400">
</p>

### 完成提醒

计时结束后显示阶段完成提示，并播放提示音。

<p align="center">
  <img src="assets/images/done.png" alt="完成提醒" width="400">
</p>

## 操作概览

| 操作       | 说明                           |
| ---------- | ------------------------------ |
| 方向键     | 切换页面（工作 / 休息 / 设置 / 主题） |
| `E`        | 开始计时 / 切换主题            |
| `E`（长按）| 在设置页恢复默认设置           |
| `C` / `V`  | 调整设置 / 重置计时            |
| `ESC`      | 退出选中状态                   |
